# ☕ Coffee Sales Analysis

This is my second data analysis project. It analyzes coffee bean sales data to uncover trends, customer segments, and sales performance. The dataset includes information about orders, products, and customers.

## 📌 Project Goals

- Identify coffee consumption trends based on coffee type and customer location.
- Find the most profitable coffee types and give insights to improve sales.
- Analyze which coffee products are ordered the most and which are ordered the least.
- Try to segment customers using the information from the databases.

## 🧪 Steps of Analysis

1. **Data Cleaning** – Remove duplicates, fix missing values, and standardize formats.
2. **Data Integration** – Use SQL to join tables (orders, products, customers).
3. **Trend Analysis** – Study sales over time and product performance.
4. **Customer Segmentation** – Use purchase data to group customer behavior.
5. **Data Visualization** – Create charts and dashboards in Excel and Power BI.

## 🧰 Tools Used

- **Excel** – Data cleaning, exploration, and basic analysis.
- **Power BI** – Data cleaning with Power Query, visualizations, and dashboards.
- **SQL** – Queries to analyze and join data from multiple tables.
